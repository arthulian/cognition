# Interaction Design

## Core Interaction Components
1. Dark Mode Toggle: Persistence via localStorage.
2. Skill Cards Grid: Hover (lift/scale), Click (expand).
3. Skill Detail Modal: Previous/Next navigation.
4. Category Filter: Smooth state transitions.
5. Progress Indicator: Subtle visual tracking.

## Accessibility
- WCAG AA contrast, Keyboard nav, Reduced motion support.
