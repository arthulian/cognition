# Design Style Guide - Cognitive Capacities Website

## Design Philosophy
### Color Palette
- **Primary**: #1a1a1a / #ffffff
- **Accent**: #f59e0b
- **Secondary**: #6b7280
- **Dark Mode**: #000000

### Typography
- **Display**: "Playfair Display"
- **Body**: "Inter"
- **Monospace**: "JetBrains Mono"

## Visual Effects & Styling
### Used Libraries
- Anime.js, Splitting.js, p5.js, ECharts.js, Pixi.js

### Animation & Effects
- Card Hover, Text Reveal, Background Particles, Transitions

### Layout Principles
- 12-column grid, 8px base spacing, Responsive (768px, 1024px)
